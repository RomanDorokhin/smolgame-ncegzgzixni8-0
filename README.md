# Metamorphosis Game 🧬✨

Игра о циклах, перерождении и постоянной трансформации. Создана для Telegram WebApp.

## 🚀 Как запустить локально

Так как игра использует **ES-модули**, её нельзя просто открыть двойным кликом по `index.html`. Нужен локальный сервер.

### Вариант 1 (Python) - Самый простой
Если у вас установлен Python, выполните в терминале:
```bash
npm start
```
Затем откройте: **http://localhost:8080**

### Вариант 2 (Node.js)
Если у вас установлен Node.js:
```bash
npm run dev
```

## 🎮 Управление
- **Клавиатура**: Стрелки / WASD / Пробел
- **Мобильные**: Свайпы и Тапы (адаптивная логика для каждого режима)
